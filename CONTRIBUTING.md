# Contributing Guidelines

## Welcome!
Thank you for considering contributing to the Shipping Calculator project!

## How to Contribute
1. Fork the repository
2. Create a new branch for your feature
3. Make your changes
4. Submit a pull request

## Code Standards
- Write clean, readable code
- Add comments where necessary
- Test your changes before submitting

## Reporting Issues
- Use the GitHub Issues tab
- Describe the problem clearly
- Include steps to reproduce the issue

## Pull Request Process
1. Ensure your code follows our standards
2. Update the README if needed
3. Wait for review and approval from maintainers

## Attribution
All contributors will be recognized in the README file.
